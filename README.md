## 📊 Supermarket Sales Data Analysis Project
I completed a data analysis project using a **supermarket sales dataset** to explore business performance and customer behavior. The project demonstrates the full data analysis workflow, from data collection to visualization and insight generation.

### 🔍 Key Steps in the Project:
✔ **Data Collection**
* Used a structured dataset containing sales transactions, customer details, product categories, and payment methods.

✔ **Data Cleaning & Preprocessing**
* Handled missing values and duplicates
* Converted date columns into proper datetime format
* Created new features such as month and day for deeper analysis

✔ **Exploratory Data Analysis (EDA)**
* Analyzed total sales and transaction trends
* Identified top-performing product categories
* Studied sales distribution across cities and customer types
* Explored customer preferences based on gender and payment methods

✔ **Data Visualization**
* Built interactive and clear visualizations using tools like **Matplotlib/Seaborn and Power BI**
* Included:
  * Bar charts (Sales by Product Line, City)
  * Line charts (Monthly Sales Trends)
  * Pie/Donut charts (Payment Methods, Gender Distribution)
  * KPI metrics for business performance

### 📈 Key Insights:
* Certain product categories contribute significantly to overall revenue
* Sales vary across cities, highlighting location-based performance differences
* Customers show strong preference for digital payment methods
* Seasonal patterns observed in monthly sales trends

### 🎯 Tools & Technologies:
* Python (Pandas, Matplotlib, Seaborn)
* Power BI
* Data Cleaning & Analysis Techniques

### 💡 Outcome:
This project helped me strengthen my skills in **data cleaning, analysis, visualization, and storytelling**, and provided actionable insights that can help businesses make data-driven decisions.
